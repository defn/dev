Tutorial : TOC

  • Configure kubeconfig for in-cluster Kubernetes
    cp ~/.kube/config-in-cluster ~/.kube/config
  • Run Tilt to build and deploy the shell-controller. Press space to open Tilt web UI.
    tilt up
  • Open the Tilt web UI by pressing the space bar while in the shell panel.
    Press s to stream logs. This mode will no longer open the web UI.
  • Edit hook.sh file to change the shell-operator
    Tilt will Live Update in the "hook" resource.
  • Edit deploy.yaml file to change Kubernetes deployment.
    Then restart the (Tilt) resource in Tilt web UI.
  • Run a comand in the operator pod.
    kubectl exec -ti deploy/hook -- /shell-operator queue list
